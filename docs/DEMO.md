## 2. 上传图片

![图片加载完成](../demo/02_image_loaded.png)

上传后图片显示在画布中央，左侧工具栏变为可操作。

**上传方式：**
- 📁 选择图片 — 单张上传
- 📂 批量图片 — 多张同时上传
- 🎬 视频文件 — 上传视频提取关键帧
- 📷 摄像头 — 实时视频流

---

## 3. 点击分割

![添加标记点](../demo/03_points_added.png)

在物体上点击添加标记点，SAM 推断目标物体。

**操作：**
1. 选中「📌 点击」模式
2. 左键点击物体（正样本，绿色圈）
3. 右键点击排除区域（负样本）
4. 点击「✂️ 分割」

**技巧：** 点击物体中心效果最好；`Ctrl+Z` 撤销标记点

---

## 4. 框选分割

矩形框框选目标区域，SAM 在框内精确分割。

**操作：** 切换「📦 框选」→ 拖拽画框 → 点击「✂️ 分割」

---

## 5. 画笔编辑

分割后用画笔微调掩码边缘。

**操作：**
1. 先执行一次分割得到掩码
2. 切换「🖌️ 画笔」模式
3. 选「➕ 添加」或「➖ 擦除」
4. 调节画笔大小（5-50px）
5. 在画布上涂抹修改

`Ctrl+Z` 撤销 | `Ctrl+Y` 重做

## 6. 自动检测

![自动检测结果（ViT-L）](../demo/10_auto_detect_vitl.png)

点击「🔍 自动检测」，系统自动扫描整张图片。

**检测结果：**
- 类别标签（如 orange 橙子）
- 置信度百分比
- 边界框坐标
- 彩色叠加区域

**工作原理：** 网格采样 → SAM 候选掩码 → 去重合并 → ResNet 识别 → 过滤低置信度

---

## 7. 自动分割

点击「✨ 自动分割」，生成所有物体的精确分割掩码叠加图。

**与自动检测的区别：**
- 自动检测 → 检测列表 + 标注框
- 自动分割 → 完整的彩色掩码叠加图

## 8. 图像识别

![识别结果（ViT-L）](../demo/11_recognize_vitl.png)

点击「🏷️ 识别」，ResNet50 分析图片内容。

**识别结果：**
- Top 分类预测：如 **orange 橙子 (75.2%)**
- 置信度进度条（彩色标识）
- 场景分析：亮度 / 色调 / 场景类型

> 使用 ViT-L 后识别从 dining table (65.2%) 提升到 orange 橙子 (75.2%)

## 12. 自定义训练

![训练界面](../demo/23_training_section.png)

用自己数据训练专属分类器。

### 训练流程

1. 添加类别 — 输入名称（如：苹果、橙子）→ 点「添加」
2. 收集样本 — 分割物体 → 点对应类别按钮添加样本
3. 开始训练 — 点「🎯 开始训练」（ResNet18 微调，10 epoch）
4. 自动生效 — 自定义分类器接管识别

**功能按钮：**
- 🎯 开始训练 — 单次训练
- 📂 批量训练 — 多类别批量
- 📊 评估模型 — 查看准确率/混淆矩阵
- 💾 导出 — ZIP 包（模型+类别+元数据）
- 📥 导入 — 从 ZIP 恢复

**数据增强（默认开启）：** 翻转 / 旋转 / 色彩抖动 / 随机裁剪 / 随机擦除

## ViT-B vs ViT-L 对比

同一张图片（groceries.jpg）的识别差异：

| | ViT-B | ViT-L |
|---|---|---|
| 模型大小 | 375MB | 1.2GB |
| 自动检测 | dining table 餐桌 | **orange 橙子** |
| 图像识别 | dining table (65.2%) | **orange (75.2%)** |
| 分割精度 | 一般 | **更精细** |

**结论：** ViT-L 在识别准确率上有显著提升，推荐使用。
