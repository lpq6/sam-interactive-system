## 3. 点击分割

![添加标记点](../demo/03_points_added.png)

在物体上点击添加标记点，SAM 根据点击位置推断目标物体。

**操作步骤：**
1. 确保左上角选中「📌 点击」模式
2. 在目标物体上**左键点击**（正样本，绿色圈）
3. 可多次点击添加更多正样本
4. **右键点击**添加负样本（排除不需要的区域）
5. 点击「✂️ 分割」按钮

**技巧：**
- 点击物体中心效果最好
- 分割不精确时加负样本修正
- `Ctrl+Z` 撤销最后一个标记点

---

## 4. 框选分割

用矩形框框选目标区域，SAM 在框内做精确分割。

**操作步骤：**
1. 切换到「📦 框选」模式
2. 在画布上按住鼠标拖拽画矩形框
3. 松开后点击「✂️ 分割」

**适用场景：** 物体边界清晰、与背景对比明显的区域

---

## 5. 画笔编辑

分割后可以用画笔微调掩码边缘。

**操作步骤：**
1. 先执行一次点击/框选分割得到掩码
2. 切换到「🖌️ 画笔」模式
3. 选择「➕ 添加」或「➖ 擦除」
4. 调节画笔大小（5-50px 滑块）
5. 在画布上涂抹修改掩码

**快捷操作：**
| 按键 | 功能 |
|------|------|
| `Ctrl+Z` | 撤销上一笔 |
| `Ctrl+Y` | 重做 |

## 6. 自动检测

![自动检测结果](../demo/05_auto_detect.png)

点击「🔍 自动检测」，系统自动扫描整张图片，检测并识别所有可见物体。

**检测结果面板显示：**
- 每个物体的**类别标签**（如 sweet potato、cucumber）
- **置信度**百分比
- **边界框**坐标
- 彩色叠加显示检测区域

**工作原理：**
1. 网格采样（默认 24×24 点）
2. 每个点用 SAM 生成候选掩码
3. 去重（IoU > 0.5 的重复检测合并）
4. 用 ResNet50 识别每个区域的物体类别
5. 过滤低置信度和泛化标签

---

## 7. 自动分割

点击「✨ 自动分割」，生成所有物体的精确分割掩码叠加图。

**与自动检测的区别：**
- **自动检测** → 返回检测列表 + 标注框
- **自动分割** → 返回完整的彩色掩码叠加图

**结果面板显示：** 检测到的区域数量（如"13 个区域"）

## 8. 图像识别

![识别结果](../demo/07_recognize.png)

点击「🏷️ 识别」，使用 ResNet50（ImageNet 预训练）分析图片内容。

**识别结果包含：**
- **Top 分类预测**：如 "shopping basket 购物篮 (85.4%)"
- 置信度进度条（彩色标识）
- **场景分析**：
  - 亮度：light
  - 色调：warm
  - 场景：outdoor

**使用方式：**
- 直接点「识别」→ 识别整张图
- 先分割某个区域再点「识别」→ 识别该区域

## 9. 彩色物体提取

![彩色物体提取](../demo/06_color_extract.png)

点击「🎨 提取彩色物体」，一键提取图中所有独立物体，每个保存为透明背景 PNG。

**提取流程：**
1. 网格密集采样（32×32 点）
2. SAM 生成候选掩码
3. ResNet50 分类 + top-5 备选标签遍历
4. 去重合并（IoU > 0.5）
5. 过滤低置信度（默认 ≥ 30%）和泛化标签
6. 每个物体裁剪为透明背景 PNG

**结果面板：**
- 物体缩略图 + 标签 + 置信度
- 置信度颜色：🟢 ≥ 80% / 🟡 ≥ 60% / 🔴 < 60%
- 点击缩略图查看详情
- 「下载」保存单个物体 / 「下载全部」打包

## 11. 模型管理

SAM 模型切换（工具栏顶部下拉框）：

| 模型 | 参数量 | 大小 | 推理速度 | 分割精度 | 适用场景 |
|------|--------|------|----------|----------|----------|
| ViT-B | 93M | 375MB | ⚡ 快 | ★★★ | 日常使用 |
| ViT-L | 308M | 1.2GB | ⏱ 中 | ★★★★ | 高精度需求 |
| ViT-H | 636M | 2.4GB | 🐢 慢 | ★★★★★ | 极致精度 |

**切换方式：** 下拉框选择 → 等待模型加载 → 自动生效

**分类模型：** ResNet50（ImageNet 1000类）始终在线，用于物体识别

## 12. 自定义训练

用你自己的数据训练专属分类器，替代通用 ResNet50。

### 训练流程

```
步骤 1: 添加类别
  输入类别名（如：猫、狗、汽车）→ 点击「添加」

步骤 2: 收集样本
  对图片进行分割 → 点击对应类别按钮
  每个类别至少 2 个样本（建议 10+ 个）

步骤 3: 开始训练
  点击「🎯 开始训练」
  模型: ResNet18 微调 | Epoch: 10 | 学习率: 0.001
  自动启用数据增强（翻转、旋转、色彩抖动、随机擦除）

步骤 4: 使用
  训练完成后自动接管所有识别功能
  无法识别时回退到通用 ResNet50
```

### 模型评估

点击「📊 评估模型」查看：
- 整体准确率
- 各类别精确率 / 召回率 / F1
- 混淆矩阵

### 模型导出/导入

- **导出**：ZIP 包（模型权重 + 类别列表 + 元数据）
- **导入**：从 ZIP 恢复
- 用途：备份 / 跨机器迁移 / 团队分享

## 13. 标注导出

分割完成后导出为标准格式：

### COCO 格式
```json
{
  "images": [{"id": 1, "file_name": "xxx.jpg", "width": 800, "height": 600}],
  "annotations": [{"id": 1, "image_id": 1, "category_id": 1, "segmentation": [...], "area": 12345, "bbox": [x,y,w,h]}],
  "categories": [{"id": 1, "name": "object"}]
}
```

### YOLO 格式
```
# labels/xxx.txt
0 0.512 0.334 0.215 0.448
1 0.780 0.612 0.105 0.220
```
格式：`class_id center_x center_y width height`（归一化坐标）

### JSON / CSV
包含分割 ID、标签、置信度、边界框、面积等信息
