# SAM Interactive System - 功能介绍与使用教程

## 系统概述

SAM Interactive System 是一个基于 Segment Anything Model (SAM) 的智能图像分割与识别系统。它集成了 SAM（图像分割）和 ResNet（物体识别），提供从交互式标注到自动检测、从单张处理到批量操作的完整工作流。

**核心能力：**
- 🎯 精确分割：点击或框选即可分割任意物体
- 🏷️ 智能识别：自动识别分割区域的物体类别
- 🎨 彩色提取：一键提取图中所有彩色物体
- 🤖 自定义训练：用自己的数据训练专属分类器
- 📦 多格式导出：COCO / YOLO / JSON / CSV

---

## 快速开始

### 1. 启动系统

```bash
cd sam-interactive-system
./start.sh
```

- 前端地址：http://localhost:5173
- 后端地址：http://localhost:8000

### 2. 上传图片

打开网页后，有三种上传方式：
- **点击上传**：点击「选择图片」按钮
- **拖拽上传**：将图片文件直接拖到上传区域
- **批量上传**：点击「批量图片」一次上传多张

支持格式：JPG、PNG、WebP

## 功能详解

### 一、交互式分割

#### 1.1 点击分割（快捷键 `1`）

在图片上点击目标物体的任意位置，SAM 会自动识别并分割出该物体。

**操作步骤：**
1. 确保工具栏选中「📌 点击」模式
2. 在图片上点击目标物体
3. 可以多次点击添加更多标记点（正样本）
4. 右键点击添加负样本（排除区域）
5. 点击「✂️ 分割」按钮执行分割

**技巧：**
- 点击物体中心区域效果最好
- 如果分割不精确，添加负样本排除多余部分
- 按 `Ctrl+Z` 撤销最后一个标记点

#### 1.2 框选分割（快捷键 `2`）

用矩形框框选目标区域，SAM 在框内进行精确分割。

**操作步骤：**
1. 切换到「📦 框选」模式
2. 在图片上按住鼠标拖拽画框
3. 松开鼠标后点击「✂️ 分割」

**适用场景：**
- 目标物体边界清晰
- 需要快速分割大区域
- 物体与背景对比明显

#### 1.3 画笔编辑（掩码微调）

分割结果不够精确时，可以用画笔手动调整掩码。

**操作步骤：**
1. 先执行一次分割得到掩码
2. 切换到「🖌️ 画笔」模式
3. 选择「添加」或「擦除」模式
4. 调节画笔大小（5-50px）
5. 在画布上涂抹修改掩码

**快捷操作：**
- 撤销：`Ctrl+Z`
- 重做：`Ctrl+Y`

---

### 二、自动检测与分割

#### 2.1 自动检测（快捷键 `D`）

SAM 自动扫描整张图片，检测并识别所有可见物体。

**工作原理：**
- 在图片上按网格采样点进行分割
- 使用 ResNet50 识别每个区域的物体类别
- 过滤低置信度和重复检测
- 返回检测列表 + 彩色叠加图

**结果包含：**
- 物体标签（类别名称）
- 置信度（0-100%）
- 边界框坐标
- 分割掩码

#### 2.2 自动分割（快捷键 `S`）

类似自动检测，但额外生成每个物体的精确分割掩码叠加图。

**与自动检测的区别：**
- 自动检测：返回检测列表和框
- 自动分割：返回完整的掩码叠加图（更直观）

---

### 三、图像识别（快捷键 `R`）

使用 ResNet50（ImageNet 预训练）识别整张图片或选中区域。

**识别结果包含：**
- Top-5 分类预测
- 各类别置信度
- 场景分析（亮度、色调、场景类型）

**使用方式：**
- 上传图片后直接点击「识别」按钮 → 识别整张图
- 先分割某个区域再点识别 → 识别该区域

---

### 四、彩色物体提取（快捷键 `C`）

一键提取图片中所有彩色物体，每个物体保存为独立的透明背景 PNG。

**操作：**
1. 上传图片
2. 点击「提取彩色物体」
3. 系统自动扫描并提取所有物体
4. 在右侧面板查看物体列表
5. 点击任意物体缩略图查看详情
6. 点击「下载」保存单个物体

**参数说明：**
- 置信度阈值：默认 30%，低于此值的物体不显示
- 最小区域面积：默认 500 像素，太小的区域会被过滤
- 去重阈值：重叠度 > 50% 的掩码自动合并

**结果面板：**
- 置信度颜色标识：绿色 ≥ 80%，黄色 ≥ 60%，红色 < 60%
- 支持下载单个或全部彩色物体
- 点击「恢复原图」回到原始状态

---

### 五、批量处理

支持一次上传多张图片，批量进行检测和识别。

**操作步骤：**
1. 点击「批量图片」上传多张图片
2. 使用左侧面板的控制按钮切换图片：
   - 「⬅ 上一张」/「下一张 ➡」
   - 滑块快速跳转
3. 对每张图片执行所需操作
4. 批量导出结果

---

### 六、视频与摄像头

#### 6.1 视频帧分割

1. 点击「视频文件」上传视频
2. 系统自动提取关键帧（最多 100 帧）
3. 使用帧控制滑块切换帧
4. 对每帧进行分割操作

#### 6.2 摄像头实时流

1. 如果检测到摄像头，显示「摄像头」按钮
2. 点击启动实时视频流
3. 在实时画面上进行分割
4. 点击「停止摄像头」释放资源

---

### 七、模型管理

#### 7.1 SAM 模型切换

系统支持三种 SAM 模型：

| 模型 | 大小 | 速度 | 精度 | 适用场景 |
|------|------|------|------|----------|
| ViT-B | 375MB | ⚡ 最快 | ★★★ | 日常使用、快速预览 |
| ViT-L | 1.2GB | ⏱ 中等 | ★★★★ | 需要更高精度 |
| ViT-H | 2.4GB | 🐢 最慢 | ★★★★★ | 最高精度、精细分割 |

**切换方式：** 在工具栏顶部的模型选择器中点击切换。

#### 7.2 自定义分类器训练

用你自己的数据训练专属物体分类器，替代通用的 ImageNet 分类。

**训练流程：**

1. **添加类别**
   - 在「自定义训练」面板输入类别名称（如：猫、狗、汽车）
   - 点击「添加」创建类别

2. **收集训练样本**
   - 对图片进行分割
   - 点击对应类别按钮，将分割结果添加为该类别的训练样本
   - 每个类别至少需要 2 个样本（建议 10+ 个）

3. **开始训练**
   - 点击「开始训练」
   - 使用 ResNet18 预训练模型微调
   - 默认 10 个 epoch，学习率 0.001

4. **使用自定义模型**
   - 训练完成后自动生效
   - 自动检测、识别等功能会优先使用自定义分类器
   - 如果自定义分类器无法识别，回退到 ResNet50

**数据增强（默认开启）：**
- 随机裁剪、翻转、旋转
- 颜色抖动、仿射变换
- 随机擦除
- 每个样本自动生成 2 个增强版本

**模型评估：**
- 点击「评估模型」查看准确率、混淆矩阵
- 各类别精确率、召回率、F1 分数

**模型导出/导入：**
- 导出：保存为 ZIP 包（模型权重 + 类别列表 + 元数据）
- 导入：从 ZIP 包恢复模型
- 用途：备份、迁移、分享

### 八、标注导出

分割完成后，可以导出为标准机器学习格式。

#### COCO 格式
- 标准 JSON 格式
- 包含 images、annotations、categories
- 可直接用于 COCO 数据集训练

#### YOLO 格式
- 标准 TXT 格式
- 归一化坐标：`class_id center_x center_y width height`
- ZIP 包含标注文件 + 类别文件 + 原图

#### JSON / CSV 格式
- 包含分割 ID、标签、置信度、边界框、面积
- CSV 使用 UTF-8 BOM 编码（兼容 Excel）

## 提升识别准确率的方法

### 当前识别机制的局限

当前系统使用 **ResNet50 ImageNet 预训练模型** 进行物体识别。ImageNet 包含 1000 个常见类别（动物、车辆、家具等），但对以下场景效果有限：

- 专业领域物体（医疗、工业、农业）
- 细粒度分类（不同品种的花/鸟）
- 中文场景常见物体
- 小物体或遮挡物体

### 方法一：切换更大的 SAM 模型

**效果：分割更精确 → 识别更准**

| 当前 | 升级后 |
|------|--------|
| ViT-B (默认) | ViT-L 或 ViT-H |

ViT-H 的分割掩码比 ViT-B 精确得多，尤其是边缘和小物体。更精确的掩码意味着更干净的裁剪区域，ResNet 识别自然更准。

### 方法二：训练自定义分类器

**效果：针对你的场景，准确率可提升 20-50%**

这是最有效的提升方式：

1. 收集你常用类别的图片（每个类别 10-50 张）
2. 用 SAM 分割出目标物体
3. 添加为训练样本并训练
4. 自定义分类器会自动接管识别

**建议：**
- 每个类别至少 10 个样本
- 样本尽量多样化（不同角度、光照、背景）
- 开启数据增强（默认开启）
- 类别不要太多（5-20 个效果最好）

### 方法三：调整采样密度

当前自动检测使用 `points_per_side=32` 的网格采样。如果图中物体很多或很小，可以增加采样密度。

在 API 调用中设置更大的 `points_per_side`：
```json
POST /api/detect/auto
{
  "image_id": "xxx",
  "points_per_side": 48,
  "min_mask_region_area": 300
}
```

### 方法四：降低置信度阈值

如果检测结果太少，可以降低阈值找回更多物体：

```json
POST /api/extract/all?image_id=xxx&min_confidence=0.2
```

### 方法五：多次检测取并集

对同一张图片使用不同的 `points_per_side` 值（如 24、32、48）进行多次检测，取结果的并集，可以覆盖更多物体。

## 常见问题

**Q: 分割速度很慢？**
A: 默认使用 ViT-B 模型，如果切换到 ViT-L/H 会变慢。建议使用 GPU（CUDA）加速。检查后端启动日志是否显示 `[GPU] Using GPU`。

**Q: 识别结果不准？**
A: 通用 ResNet50 只认识 ImageNet 的 1000 类。如果你的物体不在其中，建议训练自定义分类器。

**Q: 彩色物体提取只返回少量结果？**
A: 置信度阈值可能太高。通过 API 降低 `min_confidence` 参数（如 0.2）。

**Q: 批量上传后图片不显示？**
A: 使用左侧面板的「⬅ 上一张 / 下一张 ➡」按钮切换，或用滑块跳转。

**Q: 摄像头无法打开？**
A: 在 WSL 环境下摄像头支持有限。确保系统有可用摄像头设备，且浏览器授予了摄像头权限。
